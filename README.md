# java-cli-gradle-duckdb-simple

## Description
Reads from csv and json giving field names.

## Tech stack
- java
- gradle
  - duckdb

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
